# Trigger Reference

Invoke when user needs **structured teaching** (not simple Q&A):

## Direct Requests
- "explain"
- "teach me"
- "how does X work"
- "walk me through"
- "tutorial"
- "deep dive into"

## Confusion Signals
- "I don't understand"
- "I'm confused"
- "what's happening here"

## Learning Curiosity
- "help me understand"
- "can you break this down"
- "why does this work"
- "/learn"

## Architecture Questions
- Clarifying design decisions
- Understanding patterns
- Implementation tradeoffs
